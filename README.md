# Mandelbrot
Dyalog APL-based Mandelbrot Set Explorer
Documentation can be found [here](https://bpbecker.github.io/Mandelbrot)



## Disclaimer
The code, while somewhat reasonably organized, is nonetheless a bit of a mess. There are globals and semi-globals that are used to maintain state. My initial goal was to learn Dyalog's GUI features and not to necessarily to write exemplary code.
